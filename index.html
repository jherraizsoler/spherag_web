Clasificación de suelo con sensores Sentek (Spherag)

¿Pueden los datos de humedad, temperatura y conductividad del suelo identificar el tipo de suelo sin análisis de laboratorio?

Jorge Herraiz Soler Ralph Berrio Quispe Ignacio Buey Ruiz

¿Por qué?

Identificar el suelo por satélite o experiencia empírica es impreciso. Un estudio científico es caro. ¿Pueden los datos del sensor sustituirlo?

¿Para qué?

Optimizar el riego, reducir costes y mejorar la rentabilidad del cultivo a partir de datos que el agricultor ya tiene instalados.

WRB · clasificación internacional

Hipótesis

«Es posible identificar el tipo de suelo a partir de los datos del sensor Sentek.»

Técnicas aplicadas

  • K-Means — clustering no supervisado de lecturas
  • PCA — reducción dimensional (VWC, temp, EC)
  • Pearson / Spearman — correlaciones sensor↔clase
  • Kruskal-Wallis — diferencias sig. por profundidad
  • Regresión Logística — clasificación WRB (LOSO)
  • Random Forest Classifier — clasificación WRB (LOSO) + calibración isotónica
  • RF Regressor + Ridge — regresión USDA multioutput → triángulo textural

Hallazgos clave

  • 📦 97.564 muestras · 2 sensores (X91, X93)
  • ⚠️ Cambisols domina 89.8% → accuracy engañoso
  • ✅ RF Classifier: F1-macro 0.199 · Acc 0.809
  • ✅ Log. Reg.: F1-macro 0.158 · Acc 0.898
  • 📉 USDA R² negativo → patrón textural no transfiere entre sensores sin recalibración
  • 🔬 Kruskal-Wallis confirma diferencias significativas en VWC por capa y clase
