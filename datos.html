Pregunta de investigacion

¿La señal multinivel permite inferir el suelo?

Clasificacion WRB

Predecir soil_type_1 desde perfiles verticales de humedad y temperatura.

Regresion USDA

Estimar clay/sand/silt para derivar clases texturales USDA.

Inventario

Fuentes de datos

Parquet

21 archivos en data/logs con telemetría multinivel por canal.

Etiquetas WRB

data/sensor_data_with_type.csv con vigencia temporal por sensor.

Propiedades USDA

data/sensor_with_properties.csv con clay/sand/silt por profundidad.

Proceso

Pipeline reproducible

  1. Filtrado de bloques SENTEK y eliminación de DISCONNECTED.
  2. Enlace temporal: Imei + Code + rango de validez.
  3. Expansión por capas y canales: VWC, TEMP, VIC, voltage.
  4. Generación de WRB, USDA, Unified y ARFF.

Validacion

Consistencia de artefactos

  • WRB: 44,011 filas
  • USDA: 44,074 filas
  • Unified: 97,564 filas
  • WRB CSV = WRB ARFF
  • Unified etiquetado = WRB
  • Unified targets completos = USDA
